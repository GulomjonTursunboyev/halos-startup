```

│  1. AD IMPRESSION (10,000)                   │
│     ↓                                        │

│  4. REGISTRATION (60%) = 126-210             │
│     ↓                                        │

│     ↓                                        │
│  6. PRO CONVERSION (15%) = 8-13 paid users  │
└─────────────────────────────────────────────┘

CAC (Customer Acquisition Cost):
- Ad spend: $100
- PRO users: ~10
- CAC: $10/user

LTV (Lifetime Value):
- Monthly: 29,990 UZS (~$2.5)
- Average retention: 6 months
- LTV: ~$15

👉 LTV/CAC = 1.5x (sog'lom)
```
